PUBBLICAZIONE GRATIS (GitHub Pages)
- Crea un repo su GitHub, es: gastonco-site
- Carica TUTTI i file (index.html deve stare nella root)
- Settings → Pages → Deploy from a branch → branch: main / root → Save
- Dopo 1-2 minuti il sito è online.

Nota: è un sito “statico”, quindi il form contatti usa mailto (si apre il tuo client di posta).
